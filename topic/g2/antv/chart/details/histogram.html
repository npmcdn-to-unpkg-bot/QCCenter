直方图

    直方图的简介

    直方图，形状类似柱状图却有着与柱状图完全不同的含义。直方图关注数据的分布。标准的直方图适用于观察一段时间内连续数据的分布。在直方图中每一个矩形的高度代表这个矩形封箱内数据元的频数（数量除以矩形宽度），矩形的面积表示封箱内数据元的数量。

    通过直方图可以观察和估计哪些数据比较集中，异常或者孤立的数据分布在何处。同时直方图可以用于表示数据的一个大概的概率分布情况。

    英文名：Histogram

    $.getJSON('/data.json?filename=diamond',function (data) { var Stat = G2.Stat; var data = data.slice(800,1800); var chart = new G2.Chart({ id: 'c2', width: 400, height: 200 }); chart.axis(false); chart.source(data); chart.interval().position(Stat.summary.count(Stat.bin.rect('depth'))); chart.render(); });

    直方图的构成

    标准的直方图

    图表类型 直方图
    适合的数据 两个连续数据字段
    功能 观察数据的分布情况
    数据与图形的映射 两个连续字段分别映射到横轴和纵轴。（映射到横轴的数据字段要做封箱统计。）
    适合的数据条数 暂无限制

    非标准的直方图

    非标准的直方图由以下元素构成：

    • 横轴：表示区间
    • 纵轴：表示数量
    • 矩形：高度表示数量

    直方图的应用场景

    适合的场景

    不适合的场景

    抽取的样本数量过小，将会产生较大误差，可信度低，也就失去了统计的意义。因此，样本数不应少于50个。

    直方图的扩展

    通过变换坐标轴，我们能获得新的布局的直方图。

    直方图与其他图表的对比

    直方图与柱状图

    • 柱状图是用条形的长度(横置时)表示各类别频数的多少，其宽度(表示类别)则是固定的；
    • 直方图是用面积表示各组频数的多少，矩形的高度表示每一组的频数或频率，宽度则表示各组的组距，因此其高度与宽度均有意义。
    • 由于分组数据具有连续性，直方图的各矩形通常是连续排列，而柱状图则是分开排列。
    • 柱状图主要用于展示分类数据，而直方图则主要用于展示数值型数据。
