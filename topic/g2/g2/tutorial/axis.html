坐标轴简介

坐标轴用于表示图形的位置信息，G2 中通过指定 position 属性来定位数据的位置，所以坐标轴也根据 position 属性确定:

  • position 属性中指定的字段的数量影响生成坐标轴的个数
  • position 属性中指定的字段类型，影响坐标轴的类型，连续坐标轴或者分类坐标轴
  • 在不同坐标系下会自动确认不同类型的坐标轴
chart.point().position('cut*price')
  • cut 是分类类型的字段，所以会生成一个分类的 X 坐标轴
  • price 是连续字段会生成一个连续 Y 坐标轴

坐标轴语法

chart.axis('dim', cfg); // 配置坐标轴的配置项
chart.axis('dim', false); // 不显示对应的坐标轴
chart.axis(false); // 全部不显示坐标轴
  • G2可以配置数据字段对应的坐标轴的配置项，也可以不显示对应的坐标轴

坐标系和坐标轴

不同的坐标系会生成不同的坐标轴：

  • rect 直角坐标系下会生成垂直的2个坐标轴(x,y)；
  • plus,polar 极坐标系会生成一个圆形的坐标轴用于表示角度，一个竖直的坐标轴用于表示半径；
  • theta 坐标系，只有一个坐标轴用于表示角度，但是在实践中往往不显示；

坐标轴的配置信息

chart.axis(dim, cfg)，下表展示了 cfg 中具体的参数，具体使用详见 chart.axis() API:

参数 说明
formatter 回调函数，格式化显示的文本。
tickLine 坐标点对应的轴线，null 不显示。
line 坐标轴线的配置信息，null 不显示。
labels 文本的配置信息，null 不显示文本。
title 坐标轴的标题，如果为 null 则不显示。
titleOffset 坐标轴标题距离坐标轴线的距离。
labelOffset 坐标轴文本距离坐标轴线的距离。
grid 栅格配置信息，null 不显示，目前支持的栅格类型： line 不封闭的线；polygon 封闭的多边形；circle 圆。

示例

var Stat = G2.Stat;
$.getJSON('/data.json?filename=world.geo', function(mapData){
  var userData = [];
  var features = mapData.features;
  for(var i=0; i<features.length; i++) {
    var name = features[i].properties.name;
    userData.push({
      "name": name,
      "value": Math.round(Math.random()*1000)
    });
  }

  var chart = new G2.Chart({
    id: 'c1',
    width: 800,
    height: 500,
    plotCfg: {
      margin: [50, 80]
    }
  });
  chart.source(userData);
  // X 坐标轴配置
  chart.axis('..long', {
    line: null, // 不展示 line
    tickLine: null, // 不展示 tickLine
    labels: null, // 不展示文本
    grid: {
      line: {
      stroke: '#d9d9d9',
      'stroke-width': 1,
      'stroke-dasharray': [4, 4]
      }
    } // 显示坐标轴栅格线
  });
  // Y 坐标轴配置
  chart.axis('..lant', {
    line: null,
    tickLine: null,
    labels: null
  });
  chart.coord('map', {
    projection: 'mercator',
    max: [180, 165.65], // 指定投影后最大的坐标点
    min: [-180, -67.21] // 指定投影后最小的坐标点
  });
  chart.legend(false);
  // Stat.map.region 会默认生成 ..long ..lant 两个字段
  chart.polygon()
    .position(Stat.map.region('name', mapData))
    .tooltip('name*value')
    .color('#000');
  chart.render();
});
