简介

var chart = new G2.Chart({
  id: 'c1',
  width: 1000,
  height: 500
});

一般情况下 G2 创建一个图表，只需要 4 个语句即可:

  1. 声明一个 Chart 对象；
  2. 设置数据源；
  3. 创建一个图层，指定图形类型、属性；
  4. 渲染图表。

本章我们主要详细讲解第一步：如何创建 Chart 对象。

属性

G2 的图表主要有下面6个属性：

  • id：图表生成时所在的 DOM 容器的id；
  • container: 图表的 DOM 容器，一般自动生成。如果不声明 id，可以直接使用此参数传入 DOM 节点；
  • width: 图表的宽度；
  • height: 图表的高度；
  • animate: 图表是否执行动画；
  • plotCfg: 图表绘图区域的配置信息，包括边距、边框、背景图片等信息，此属性详细的描述，可以参考图表背景
    • margin 边距可以是单个数值也可以是一个数组，[top,right,bottom,left]，类似于css 的margin属性；
    • border 边框的配置信息，默认为 null,可以通过 stroke 设置边框的颜色，fill 设置边框的填充；
    • background 图表的背景。

另外，G2 绘制出的图表是由三层 canvas 组成的：

  • 最上层画布，用于绘制 tooltip guide 和 legend；
  • 中间画布，用于绘制 geom；
  • 最下层画布，用于绘制坐标轴 axis。

上中下画布获取方式如下：

chart.get('frontCanvas'); // 获取最上层画布
chart.get('canvas'); // 获取中间层画布
chart.get('backCanvas'); // 获取最下层画布

示例

下面是一个带有边框以及背景的图表示例：

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <!-- 引入 G2 文件 -->
    <script src="https://as.alipayobjects.com/g/datavis/g2/1.2.1/index.js"></script>
  </head>
  <body>
    <div id="c1"></div>
    <script>
      var data = [
        {gender:'男',count:40},{gender:'女',count:30}
      ];
      var chart = new G2.Chart({ // 声明一个图表
        id: 'c1',
        width: 800,
        height: 400,
        plotCfg: {
          margin: 80, // 设置 margin
          border: {
            stroke: '#ddd',
            'stroke-width': 3, // 设置线的宽度
            radius: 10 // 设置圆角
          }, // 设置边框
          background : {
            fill: '#3F3F4F'
          } // 设置背景色
        }
      });

      chart.source(data);
      chart.col('count',{type: 'linear',min:0})
      chart.interval().position('gender*count').color('gender').size(60);
      chart.render();
    </script>
  </body>
</html>
