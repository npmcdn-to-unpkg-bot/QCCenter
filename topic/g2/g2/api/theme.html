Theme

概述

G2 图表的主题样式，默认提供了三种图表主题。

语法

G2.Theme.default; // 默认的主题
G2.Theme.dark; // 暗色背景使用的主题
G2.Theme.cheery; // 色彩较鲜活艳丽的主题。
