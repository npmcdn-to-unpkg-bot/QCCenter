Util

概述

G2.Util 是一个工具类，提供了如下方法：

  • 字符串处理；
  • 类型判断；
  • 继承相关；
  • 数组相关；
  • 其他。

语法

var Util = G2.Util;
Util.extend(ChildClass, FatherClass); // 继承
Util.isArray([]); // 判断是否是数组

字符串相关

Util.substitute(str, obj)Method

替换字符串中的字段，返回匹配字符串的正则表达式。

  • str： String 类型，模板字符串
  • obj: 一个 json 对象。
var str = 'M {x} {y}';
var point = {
  x: 100,
  y: 30
}
var result = G2.Util.substitute(str, point); // 返回 'M 100 30'

Util.ucfirst(str)Method

将字符串的第一个字母变成大写。

var result = G2.Util.ucfirst('test'); // 返回 Test

类型判断

Util.isObject(param)Method

判断参数是否为对象。

Util.isNumber(param)Method

判断参数是否为数字。

Util.isNumeric(param)Method

是否是数字或者数字字符串。

Util.isBoolean(param)Method

判断参数是否为布尔值。

Util.isFunction(param)Method

判断参数是否为函数。

Util.isArray(param)Method

判断参数是否为数组。

Util.isDate(param)Method

判断参数是否为日期。

Util.isNull(param)Method

判断一个对象是否是 null 或者 undefined。

数组相关

Util.toArray(value)Method

将 value 转换成数组。

Util.map(arr, fn)Method

转换数值的值。

var arr = [1, 2, 3, 4, 5];
var doubleArr = G2.Util.map(arr, function(val) {
  return val * 2;
}); // 将 arr 数组中每个数据 * 2

Util.filter(arr, fn)Method

过滤数组。

Util.each(arr | obj, fn)Method

遍历数组或者对象。

// 遍历对象
var obj = {
  a: 'test',
  b: 'test'
};
Util.each(obj, function(item, index) {
  // item 为 obj 的每一个属性： a: 'test' , b: 'test'
  // index 为索引值
});

// 遍历数组
var arr = [1, 2, 3];
Util.each(arr, function(item, index) {

});

Util.indexOf(arr, value)Method

返回 value 在数组中的索引。

Util.remove(arr, value)Method

将 value 从 arr 中删除。

Util.empty()Method

清空。

equalsArray(arr1, arr2) Method

判断数组内容是否相等。

其他

Util.guid(prefix)Method

产生唯一码。

var id = G2.Util.guid('g-chart'); // id:g-chart1

Util.wrapBehavior(obj, fn)Method

封装事件，便于使用上下文this,和便于解除事件时使用。

getWrapBehavior(obj,fn) Method

获取封装的事件。
