Graphic

概述

为了提升视觉上的可定制性，G2 抛出一部分图形属性的配置项供用户配置。

总归起来有两类：

  • 图形属性

  • 文本属性

图形属性

所有图形均支持以下几个属性

fillString

封闭图形填充色

参数
  • 纯色 #fff || #ffffff || red || rgb(255,255,255) || rgba(255,255,255,1) || hsl(255, 100%, 50%)
  • 渐变色
    • 线性渐变 'l (45) 0:#ffff00 1:#00ffff' // 一个斜向45度的渐变
    • 径向渐变 'r (0.5, 0.5, 0) 0:#fff 1:#ff0000' // 渐变在正中心发生

'fill-opacity'Number

填充透明度

参数
  • 0～1

strokeString

描边色

参数
  • 纯色 #fff || #ffffff || red || rgb(255,255,255) || rgba(255,255,255,1) || hsl(255, 100%, 50%)
  • 渐变色
    • 线性渐变 'l (45) 0:#ffff00 1:#00ffff' // 一个斜向45度的渐变
    • 径向渐变 'r (0.5, 0.5, 0) 0:#fff 1:#ff0000' // 渐变在正中心发生

'stroke-opacity'Number

描边透明度

参数
  • 0～1

'stroke-width'Number || String

边粗细

参数
  • 像素大小

'stroke-dasharray'Array

边虚线

参数
  • [a1, a2, a3……] // a1, a2 …… 分别指的是实线和空线所占像素的多少

文本属性

文本也是图形的一种，所有图形属性文本也都支持。除此之外，文本还支持配置下述属性。

fontFamilyString

字体

'font-weight'Number

字体粗细

'text-anchor'String

字体水平对齐方向

参数
  • start 左对齐
  • middle 居中对齐
  • end 右对齐
